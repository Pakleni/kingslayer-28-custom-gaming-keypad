## Features
* Per-Key RGB
* South Facing LED
* Hot-Swap Sockets
* Custom Plate
* USB-C
* 5-pin Sockets
* Plate Mount & PCB Screw-in Stabilizer support
* Top mount design
* QMK support
* Wrist Rest with magnetic attachment mechanism
* 5 degree positive tilt
